# E-commerce Back End

## Purpose
The purpose of this application is to generate a back end for an e-commerce site by using Sequelize with my MySQL database.

## Built with
* JavaScript
* Sequelize
* Insomnia
* Dotenv
* MySQL2
* Express.js
